Automatic music genre classification is a challenge in Music Information Retrieval. While our project isolates lyrical text for a binary classification (Pop vs. Metal), prior work extensively explores genre classification using audio signals, offering valuable methodological insights.

Research highlights a persistent trade-off between traditional feature engineering and modern deep learning. A comparative study on the GTZAN dataset found that classical models like Support Vector Machines (SVMs), using hand-crafted audio features, can actually outperform Convolutional Neural Networks (CNNs) in data-constrained scenarios due to their strong regularization effects \citep{mishra2025musicgenreclassificationusing}.

Advanced deep learning architectures excel when properly optimized. Recent work demonstrates that processing audio features through parallel CNNs with finely tuned hyperparameters can achieve classification accuracies exceeding 95\% \citep{LI2024e24892}. Furthermore, models designed to capture complex temporal dynamics, such as Convolutional Recurrent Neural Networks (CRNNs) have been shown to substantially outperform both classical classifiers and standard deep architectures across diverse acoustic datasets \citep{prajuli2026musicgenreclassificationcomparative}.

We can aim to use some of these insights when building our own classifiers, and also evaluate performance, given our feature set is currently restricted to lyrics as opposed to these different mediums of data provided and tested.
